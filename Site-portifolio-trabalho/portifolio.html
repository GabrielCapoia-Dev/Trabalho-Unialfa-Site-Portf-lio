Sistemas Operacionais e Redes de Computadores

Na imagem é a cima possivel ver um projeto de uma rede de computador, essa rede possui um switch principal, outros 3 switchs que funcionam como ponte para os aparelhos, a rede contendo 12 aparelhos possui duas VLAN's capazes de transferir informações entre aparelhos que estejam autirizados a acessar essa rede, basicamente funciona como um filtro, um otimo exemplo sobre isso seria um laboratório de computadores de uma universidade que esta conectado a uma rede, e os computadores da secretaria que estao na mesma rede do laboratório, as VLAN's faz com que os alunos no laboratório de computadores não tenham acesso aos arquivos de quem esta na secretaria, e vice versa.

Já nessa segunda imagem é possivel ver uma estrutura maior de uma rede, essa estrutura é a representação da rede de uma escola. Separado por vlans, os computadores com destaque em vermelho são vlans para a Secretaria da escola, os azuis são para a Direção e por fim os verdes são para os laboratórios da escola. Essa divisão é feita para restringir acesso a rede entre as maquinas, mantendo assim a segurança, um otimo exemplo disso é, caso um dos computadores do laboratório pegue um virus que infecte a rede, esse virus fica restrito aos computadores verdes, mesmo que todos os aparelhos estejam conetados no mesmo switch, oque torna mais facil de solucionar o problema.
